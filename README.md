# Legal AI Assistant

A RAG-based Legal AI Assistant built using **LangChain, Mistral, FAISS, and Streamlit**.
It retrieves legal documents from the web and provides **context-aware answers** through an interactive chat interface.

---

##  Features

* Chat-based legal Q&A
* Web-based document retrieval
* Semantic search using FAISS
* Context-aware responses using Mistral

---

##  Tech Stack

Python • LangChain • Mistral • FAISS • Streamlit

---

##  Run

```bash
pip install -r requirements.txt
streamlit run main.py
```

---

## Setup

Create `.env` file:

```env
MISTRAL_API_KEY=your_key
OPENAI_API_KEY=your_key (optional)
```
